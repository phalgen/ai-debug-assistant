# 🛠️ AI Debug Assistant

An intelligent debugging assistant that analyzes programming errors and provides explanations, fixes, and suggestions using a hybrid approach of rule-based logic and semantic similarity.

---

## 🚀 Features

- 🔍 **Error Analysis** – Identifies common Python errors (TypeError, NameError, etc.)
- 🧠 **Semantic Similarity** – Uses Sentence Transformers to match similar past errors
- ⚡ **Rule-Based Engine** – Fast heuristic-based debugging suggestions
- 💡 **Detailed Fixes** – Provides explanations and example corrections
- ⚠️ **Typo Detection** – Detects common coding mistakes (e.g., `prit → print`)
- 📊 **Confidence Score** – Indicates similarity confidence of suggested fixes
- 🗂️ **Memory Storage** – Stores past errors for reuse and learning

---

## 🏗️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** FastAPI  
- **ML Model:** Sentence Transformers (`all-MiniLM-L6-v2`)  
- **Language:** Python
